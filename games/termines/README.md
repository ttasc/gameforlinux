# 💣 Termines

**A minimalist, terminal-first Minesweeper clone written in Go**

## 📖 About

Termines is a terminal-based Minesweeper game that brings the classic puzzle experience to your command line. Fast, simple, and completely offline.

## 🎮 Features

- **Terminal-First Design:** Optimized for terminal gameplay
- **Classic Gameplay:** Familiar Minesweeper mechanics
- **Responsive Controls:** Instant, predictable responses
- **Lightweight:** Single binary, no dependencies

## 🕹️ How to Play

1. Use arrow keys to navigate the grid
2. Left-click (spacebar) to reveal cells
3. Right-click (r key) to flag/unflag suspected mines
4. Reveal all safe cells to win
5. Avoid the mines!

## 📦 Installation

```bash
make install-termines
# or
curl -fsSL https://raw.githubusercontent.com/ttasc/gameforlinux/main/install.sh | bash
```

## 🚀 Quick Start

```bash
termines
```
