# 🧩 Sudokute

**A minimalist, terminal-native Sudoku implementation written in Go**

## 📖 About

Sudokute is a clean, fast, and zero-dependency Sudoku game designed for terminal enthusiasts. Perfect for quick gaming sessions in your terminal.

## 🎮 Features

- **Minimalist Design:** Clean, distraction-free interface
- **Terminal-Native:** Fully keyboard-driven
- **Fast Performance:** Instant response times
- **Zero Dependencies:** Single binary, no external requirements

## 🕹️ How to Play

1. Navigate the grid with arrow keys
2. Enter numbers (1-9) to fill cells
3. Delete incorrect entries with backspace
4. Complete the puzzle to win

## 📦 Installation

```bash
make install-sudokute
# or
curl -fsSL https://raw.githubusercontent.com/ttasc/gameforlinux/main/install.sh | bash
```

## 🚀 Quick Start

```bash
sudokute
```
