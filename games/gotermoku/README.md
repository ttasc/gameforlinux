# ⚫ Gotermoku

**A minimal, terminal-based Gomoku game written in Go**

## 📖 About

Gotermoku is a terminal implementation of Gomoku (Five in a Row), a classic two-player board game. Play against the AI or challenge a friend.

## 🎮 Features

- **AI Opponent:** Play against intelligent AI
- **Terminal UI:** Minimalist board display
- **Two-Player Mode:** Challenge a friend locally
- **Instant Performance:** No lag or delay

## 🕹️ How to Play

1. Navigate with arrow keys
2. Place your piece with Enter/Spacebar
3. First to get 5 in a row (horizontal, vertical, or diagonal) wins!
4. Play against AI or another player

## 📦 Installation

```bash
make install-gotermoku
# or
curl -fsSL https://raw.githubusercontent.com/ttasc/gameforlinux/main/install.sh | bash
```

## 🚀 Quick Start

```bash
gotermoku
```
